# DeepSeek Monitor · 安装与使用说明书（Windows）

一个常驻系统托盘的小工具，用来随时查看你的 **DeepSeek 账户余额、今日/本月消耗、各模型 Token 用量和 7 天趋势**。

---

## 一、系统要求

- Windows 10 / 11
- 已安装 [Node.js](https://nodejs.org/)（建议 18 及以上版本）
  - 检查方法：按 `Win + R` 输入 `powershell` 回车，输入 `node -v`，能显示版本号即可。
- 一个 DeepSeek **开放平台**账户（[platform.deepseek.com](https://platform.deepseek.com/)）

---

## 二、安装与启动

### 方式 A：下载安装包（最省事，推荐普通用户）

1. 打开 [**最新版 Release**](https://github.com/chunqingjtr/DeepSeekMonitor/releases/latest)。
2. 下载 `DeepSeek.Monitor.Setup.x.x.x.exe`。
3. 双击运行安装向导（可选择安装位置）→ 完成后**桌面和开始菜单自动出现快捷方式**。
4. 双击桌面图标即可启动，无需安装 Node 环境。

> ⚠️ 安装包未做数字签名，Windows SmartScreen 可能提示"未知发布者"，点 **更多信息 → 仍要运行** 即可。

### 方式 B：从源码运行（开发者 / 想改代码）

1. 下载本项目代码：
   - 在 [仓库页面](https://github.com/chunqingjtr/DeepSeekMonitor) 点绿色 **Code → Download ZIP**，解压到任意目录，例如 `D:\DeepSeekMonitor`；
   - 或用 git 克隆：
     ```powershell
     git clone https://github.com/chunqingjtr/DeepSeekMonitor.git D:\DeepSeekMonitor
     ```
2. 打开 PowerShell，进入目录并安装依赖（只需第一次）：
   ```powershell
   cd D:\DeepSeekMonitor
   npm install
   ```
3. 启动：
   ```powershell
   npm start
   ```
   启动后窗口不会立刻弹出，程序以**图标形式常驻在屏幕右下角系统托盘**（时间旁边、可能要点 `^` 展开）。

> 🚀 **更省事的打开方式**：项目根目录有个 **`启动.bat`**，以后**双击它**即可启动，不用每次开 PowerShell。
> 也可以右键 `启动.bat` → 发送到 → 桌面快捷方式。

> 💡 如果你在中国大陆、访问 GitHub/DeepSeek 需要代理，请确保代理软件（如 Clash）已开启；
> 本工具访问的是 `api.deepseek.com`，国内一般可直连，无需代理。

### 方式 C：自己打包 `.exe` 安装包（进阶，可选）

Release 里的安装包就是这样打出来的。如需自行打包：
```powershell
cd D:\DeepSeekMonitor
npm install -D electron-builder
# 沙盒/AppData 受限环境下，把缓存指到真实盘符可避免 EXDEV 跨盘错误：
$env:ELECTRON_BUILDER_CACHE = "D:/eb-cache"
npx electron-builder --win
```
打包完成后，在 `dist\` 目录里会生成安装包（`DeepSeek Monitor Setup x.x.x.exe`），双击安装即可。

## 三、首次配置（填入 API Key）

1. **左键点击**托盘里的鲸鱼图标 → 弹出主面板。
2. 点主面板右上角的**齿轮 ⚙️** 进入设置。
3. 在 **API Key** 一栏填入你的 Key → 点 **「验证并保存」**。
   - 验证成功会提示「当前余额 ¥xx」，状态显示「已配置」。

### 怎么拿到 API Key？

1. 打开 [platform.deepseek.com](https://platform.deepseek.com/) 登录。
2. 左侧菜单 **API keys** → 点 **「创建 API key」**。
3. **立刻复制**弹出的完整 `sk-...`（⚠️ 只在创建时显示一次，关掉就再也看不到）。
4. 粘贴进 App。

> **重点：余额和用量是按「账户」统计的，不是按 Key 统计的。**
> 同一账户下不管有几个 Key、填哪一个，看到的都是同一份数据。
> 如果你手上没有保存过完整的旧 Key，**直接新建一个**填进去即可，不花钱、也不影响已有的 Key。

## 四、日常使用

### 托盘图标操作
| 操作 | 效果 |
|------|------|
| **左键**点图标 | 弹出 / 收起主面板 |
| **右键**点图标 | 菜单：刷新 / 设置 / 退出 |
| 鼠标悬停 | 显示当前余额 |

### 主面板内容
- **账户余额**：账户总余额，右上角绿色「在线」表示账户可用。
- **今日消耗 / 本月消耗**：当天和当月的花费。
- **V4 Flash / V4 Pro**：两个模型各自的 Token 用量和花费，右侧进度条表示占比。
- **消耗趋势**：最近 7 天的 Token 消耗柱状图。
- **点击任意模型行** → 进入该模型详情页（API 请求次数、Tokens 总量、按日 Token 消耗图），左上角「返回」回到主面板。

### 设置项说明
- **桌面小组件**：在桌面常驻一个紧凑悬浮窗（Windows 版以置顶小窗实现）。
- **开机自启**：登录 Windows 后自动在后台启动并常驻托盘。
- **自动刷新**：后台自动拉取数据的间隔，可选 30 秒 / 60 秒 / 2 分钟 / 5 分钟。
- **用量数据导入**：见下一节。

## 五、用量/趋势图是空的怎么办？

**余额一定能查到**，但 Token 用量明细接口（`/v1/usage`）DeepSeek **多数账户未开放**，会返回 404 —— 这时面板只显示余额，用量和趋势图为空，**属于正常现象**（macOS 原版也是如此）。

解决办法（回退数据源）：
1. 在 DeepSeek 平台「用量信息」页导出官方用量文件（CSV）。
2. App → 设置 → **用量数据导入** → 选择该 CSV / JSON 文件。
3. 导入后面板的用量、各模型 Token、趋势图就会用这份数据填充。

支持的列名（CSV 表头，自动识别）：`date / model_name / total_tokens / cost_in_cents / request_count`（中英文表头均可）。

## 六、常见问题（FAQ）

**Q：填了 Key，余额却是 ¥0？**
A：说明这个**账户在 API 平台没充过值**。本工具查的是 **API 平台**的充值余额，跟网页版 / App 的会员订阅是两套独立的账。只用网页版聊天、没在 platform 平台充值的话，这里就是 0，属正常。

**Q：验证失败 / 提示 401？**
A：Key 复制不完整或已失效。重新创建一个 Key 再填。注意首尾不要带空格。

**Q：网络错误 / 连接超时？**
A：检查网络；若用代理软件，确认其正常工作。本工具直接访问 `api.deepseek.com`。

**Q：怎么彻底退出？**
A：右键托盘图标 → 退出。（直接关主面板只是收起，不会退出。）

**Q：退出后怎么重新打开？**
A：任选一种 ——
① 双击项目里的 **`启动.bat`**；
② 双击**桌面快捷方式**「DeepSeek Monitor」；
③ PowerShell 里 `cd D:\DeepSeekMonitor; npm start`。
想开机自动启动：打开后进入 设置 → 打开 **「开机自启」** 开关，以后登录 Windows 就会自动常驻托盘，无需手动打开。

**Q：数据存在哪？卸载会泄露吗？**
A：API Key 和缓存只存在本机用户数据目录
`C:\Users\<你的用户名>\AppData\Roaming\deepseek-monitor\`，
不会上传任何服务器，代码仓库里也不包含你的 Key。
